# CPO
Constrained Policy Optimization implementation on Safety Gym

## requirement
- gym
- mujoco
- safety_gym
- tensorflow 1.13.1
- pytorch 1.10.1

## how to use
- `python train.py #training`
- `python train.py test #test`

## reference
https://arxiv.org/abs/1705.10528
https://github.com/openai/safety-starter-agents
